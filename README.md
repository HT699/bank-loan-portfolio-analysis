Bank loan portfolio analysis using SQL, LibreOffice and Power BI 

Bank Loan Portfolio Analysis
Tools: SQL (MySQL) | LibreOffice Calc | Power BI  
Domain: Finance / Credit Risk Analysis  
Dataset: 38,576 bank loan records.

Project Overview:
Analyzed a portfolio of 38,576 bank loans to identify default risk drivers and portfolio performance trends.
Built an interactive two-page Power BI dashboard presenting findings to a finance executive audience.

Business Questions Answered
1. What is the overall loan repayment and default rate?
2. Which loan grades carry the highest default risk?
3. Which loan purposes have the worst repayment rates?
4. How has loan volume trended month over month?
5. Does debt-to-income ratio predict default likelihood?

Key Findings
- 13.82% overall default rate across 38,576 loans.
- Grade G loans default at 31.31% nearly 1 in 3.
- Small business loans carry highest default risk at 25.62%.
- Default rate rises consistently from 11.96% to 15.56% as DTI increases.
- Loan volume showed consistent upward trend throughout 2021.
